BriteCorePOC
========================================================================
BriteCorePOC project featuring an session authentication to create and view RiskType(s) and RiskInstances (s) based on RiskTypes.
* [BriteCorePOCUI](BriteCorePOCUI)
* [BriteCorePOCAPI](BriteCorePOCAPI)

#### Dependencies
- python 3.5+
- pip
- node 8.9.0
- npm 5.6.0
